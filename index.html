Collapse

Oculta o muestra, es decir, expande o colapsa un elemento

Para que esto ocuura tiene que 2 elementos importantes 1.- El elemento disparador que puede ser un boton, enlace, algun elemento html y 2.- El elemento de destino es decir aquel elemento a la que vamos apuntar

Los botones van a ser los elementos disparadores y el bloque es el elemento de destino
