Barra de navegacion

Para implementar el menu sandwich se aplica la clase '.navbar-toogler' a la etiqueta 'button'. este va ha ser el elemnto disparador a la lista del menu. click en sandwich y sale lista de menu

data-toggle="collapse" le asigna al sandwich como el elemento disparador

data-target="#menu-principal" es a donde apunto el disparador

aria-controls="menu-principal" permite al lector de pantalla saltar directamente al contenido manipulado

El elemento victima está expandido? la respuesta es NO (false). aria-expanded="false"

aria-label="Desplegar menu de navegacion"cuando un usuario cieguito navega atravez de los elementos cuando el foco este en este sandwich, el lector de pantalla de va ha decir: 'desplegar menu de navegacion'

con span y '.navbar-toogler-icon aparecen las 3 lineas del sandwich

Las 3 lineas del icono son claros (casi blanco) porque le aplicamos la clase '.navbar-dark'. El fondo ambién tiene un gris claro. Con '.navbar-light' las lineas se hacen negras
