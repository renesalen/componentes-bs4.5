Barra de navegacion

Para la barra de navegacion existe la etiqueta 'nav'. No es recomendable utilizar otras etiquetas (div, container,etc).

Viendo en inspeccionar elemento, vemos que 'nav' es un contenedor flexbox que tiene un 'justify-content: space-between

2. definimos el tema del menu que fondo va ha tener

Cuando pensamos tener un 'nav' oscuro entonces utilizar navbar-dark para que programe un color claro de texto y luego agregar un color personal con 'style'

3. definimos lo colores con la etiqueta bg-xxx que viene con bootstrap

El '.container' que está dentro de la etiqueta 'nav' se ha convertido en un flexbox, esta clase .container' ya biene preparado para asumir estos cambios. Al inspecionar elemento podemos ver que tiene 'display:flex' y 'justify-content: space-between'

Entonces cuando nosotros querramos cambiar la alineación de estos elementos que conforman nuestra barra vamos a tener que aplicar las clases al container, ya este es el 'Padre' de los logotipos, los textos de enlace y otros del navbar. Si no utilizariamos '.container' entonces el padre sería la etiqueta 'nav'.

5. El logotipo va con la clase 'navbar-brand' lo que lo hace responsive

Si queremos agrandar el tamaño del logo brand poner el texto del logo dentro de un 'span' el cual es el hijo de 'a'. Al 'span' se le asigna la clase '.h4' o similar

Cuando uno quiere agregar imagen para el logo pegar la imagen dentro de la etiqueta 'a' con su clase '.navbar-brand'
